Руководство пользователя "ames.py"

Программа является кроссплатформенным скриптом, написанном на языке Python версии 3.11.

Программа генерирует выкройки комнаты Эймса: пол, потолок и стены в векторном формате SVG.

Чтобы получить выкройки, нужно найти в тексте программы и задать координаты ключевой точки E (см. рисунок ) а также её размеры:

E = [-20, -20, 0]

WIDTH  = 40
HEIGHT = 40
DEPTH  = 40

Размеры и координаты измеряются в миллиметрах. Также важный параметр --- это масштаб глубины комнаты Эймса:

SCALE = 2

При необходимости также можно поменять коодринаты точки зрения V, которая по умолчанию находится в начале координат:

V = [0, 0, 0]

Если вы хотите другие узоры, внимательно прочитайте комментарии в тексте программы и сделайте их самостоятельно.

Файлы сгенерированных выкроек находятся в каталоге 'output/svgs/*', который должен находится в том же месте, что и скрипт самой программы. Эсли этот каталог отсутствует, создайте его.


Руководство пользователя "ames-img.py"

Программа является кроссплатформенным скриптом, написанном на языке Python версии 3.11.

Программа генерирует искаженные картинки узоров для выкроек комнаты Эймса, которые создаются программой "ames.py".

Настройки программы аналогичны настройкам программы "ames.py". Дополнительно вам нужно разобраться как генерируются узоры и задать пути к файлам исакажаемых изображений. По умолчанию файлы примеров располагаются в подкаталоге 'output/pictures/'

Файлы сгенерированных изображений находятся в по подкаталоге 'output/pictures/*' каталога скрипта самой программы. Эсли этот подкаталог отсутствует, создайте его.


Руководство пользователя "img-projection.py"

Программа является кроссплатформенным скриптом, написанном на языке Python версии 3.11.

Это упрощенная версия программы "ames-img.py". Она не имеет никакого отношения к комнате Эймса и просто генерирует проекцю картинки на плоскость через заданную точку зрения.
